How Decision Trees Work

Decision trees make predictions by asking a series of yes/no questions about the features.

  1. Start at the root node
  2. Evaluate the splitting condition
  3. Follow the branch based on the answer
  4. Repeat until reaching a leaf node
  5. Return the prediction from the leaf

How Logistic Regression Works

Logistic regression uses the sigmoid function to model probabilities and create decision boundaries.

  1. Linear combination of features: z = w₀ + w₁x₁ + w₂x₂
  2. Apply sigmoid function: p = 1 / (1 + e^(-z))
  3. Use gradient descent to optimize weights
  4. Create decision boundary where p = 0.5

How SVM Works

SVM finds the optimal hyperplane that separates classes with maximum margin.

  1. Find the hyperplane that separates classes
  2. Maximize the margin between classes
  3. Identify support vectors (closest points)
  4. Use kernel tricks for non-linear separation
